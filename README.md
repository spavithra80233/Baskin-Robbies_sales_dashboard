Baskin-Robbins Sales Dashboard – Power BI Project
📌 Project Overview
This project presents an interactive Sales Dashboard built using Microsoft Power BI, analyzing sales performance data of Baskin-Robbins. The dashboard provides meaningful insights into revenue, profit, product performance, and customer trends to support business decision-making.
The objective of this project is to transform raw sales data into actionable business insights through data modeling, DAX calculations, and interactive visualizations.
🛠️ Tools & Technologies Used
📊 Microsoft Power BI
📈 DAX (Data Analysis Expressions)
📂 Data Modeling
📑 Excel/CSV Dataset
📊 Key KPIs Tracked
💰 Total Sales
📦 Total Quantity Sold
📈 Profit
📊 Sales by Category
🏪 Sales by Region
📅 Monthly Sales Trend
🏆 Top-Selling Products
📈 Dashboard Features
Interactive slicers for filtering by Date, Region, and Category
Dynamic KPIs using DAX measures
Time-based analysis using Calendar table
Category-wise and region-wise performance comparison
Clean and user-friendly dashboard layout
📂 Dataset Description
The dataset contains the following columns:
Order Date
Product Name
Category
Region
Sales Amount
Cost
Quantity
🧮 Sample DAX Measures Used
DAX
Copy code
Total Sales = SUM(Sales[Sales Amount])

Total Profit = SUM(Sales[Sales Amount]) - SUM(Sales[Cost])

Total Quantity = SUM(Sales[Quantity])
